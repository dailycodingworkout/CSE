# Time Management Mastery

## The 60-Second Rule

```
For any question type:
‚îú‚îÄ‚îÄ 0-30 seconds: Understand and identify type
‚îú‚îÄ‚îÄ 30-45 seconds: Apply strategy, eliminate options
‚îú‚îÄ‚îÄ 45-60 seconds: Select answer
‚îî‚îÄ‚îÄ Beyond 60 seconds: Mark and move on
```

## Time Traps to Avoid

| Trap | Problem | Solution |
|------|---------|----------|
| **Perfectionism** | Spending 3+ minutes on one question | Set mental timer, move on |
| **Second-guessing** | Changing answers repeatedly | Trust first instinct unless certain |
| **Reading too carefully** | Reading every word slowly | Strategic skimming for RC |
| **Blank paralysis** | Getting stuck on tough blanks | Skip, return later |
| **Order obsession** | Solving in given order | Attack easy questions first |

# Error Minimization Techniques

## Pre-Answer Checklist

Before selecting any answer, quickly verify:

```
‚òê Did I read the COMPLETE question?
‚òê Did I check what's being ASKED?
‚òê Did I eliminate OBVIOUSLY wrong options?
‚òê Does my answer FIT grammatically?
‚òê Does my answer make LOGICAL sense?
‚òê Am I SURE, or is this a guess?
```

## Common Mistake Patterns

### Mistake 1: Misreading "NOT" or "EXCEPT"

```
Question: "Which of the following is NOT correct?"
Mistake: Selecting a correct option

PREVENTION: Circle negative words in the question
```

### Mistake 2: Partially Correct Answers

```
Question asks for "best" answer
Mistake: Selecting first option that seems right

PREVENTION: Read ALL options before selecting
```

### Mistake 3: Overthinking

```
Simple question with obvious answer
Mistake: Looking for hidden complexity

PREVENTION: If it looks simple, it probably is simple
```

### Mistake 4: Vocabulary Overconfidence

```
You "know" a word but are actually wrong
Mistake: Selecting without verification

PREVENTION: Use context clues to verify meaning
```

## Section A: Grammar (Error Detection)

**Directions**: Find the error in the following sentences.

**Q1.** Neither the manager nor his subordinates (A) / was present (B) / at the meeting. (C) / No error (D)

<details>
<summary>Answer</summary>
<b>(B) was ‚Üí were</b>

With neither...nor, verb agrees with the closer subject (subordinates = plural).
</details>

---

**Q2.** Each of the students (A) / have brought (B) / their books to class. (C) / No error (D)

<details>
<summary>Answer</summary>
<b>(B) have ‚Üí has</b>

"Each" is singular and requires a singular verb.
</details>

---

**Q3.** The population of India (A) / is greater than (B) / Pakistan. (C) / No error (D)

<details>
<summary>Answer</summary>
<b>(C) Pakistan ‚Üí that of Pakistan</b>

Compare population to population, not population to a country.
</details>

---

**Q4.** I have been living here (A) / since (B) / five years. (C) / No error (D)

<details>
<summary>Answer</summary>
<b>(B) since ‚Üí for</b>

Use "for" with duration (five years), "since" with a point in time (since 2019).
</details>

---

**Q5.** The committee (A) / have decided to postpone (B) / the meeting until next week. (C) / No error (D)

<details>
<summary>Answer</summary>
<b>(B) have ‚Üí has</b> (or No error in British English)

When acting as a unit, committee takes singular verb in American English.
</details>

## Section D: Para Jumbles

**Q16.** Arrange the following sentences in correct order:

P: This has led to widespread deforestation.
Q: The global demand for palm oil has increased dramatically.
R: Many species are now facing extinction as a result.
S: Governments are implementing stricter regulations to address this issue.

A) QPRS   B) PQRS   C) QRPS   D) SPQR

<details>
<summary>Answer</summary>
<b>A) QPRS</b>

- Q introduces the cause (increased demand)
- P shows immediate effect (deforestation)
- R shows consequence ("as a result")
- S shows response (regulations)
</details>

A) BACD   B) ABCD   C) BCAD   D) CABD

<details>
<summary>Answer</summary>
<b>A) BACD</b>

- B introduces the topic
- A contrasts with "However"
- C explains the cost
- D concludes with "Therefore"
</details>

# Part 1: Word Building - The DNA of Vocabulary

## ğŸ§  The Mental Model: Words Are Like LEGO Blocks

```
PREFIX + ROOT + SUFFIX = WORD

un     +  do   +  able  = undoable
pre    +  dict +  ion   = prediction
dis    +  agree+  ment  = disagreement
```

**Why This Matters**: If you know 50 roots + 30 prefixes + 20 suffixes, you can decode 1000+ words!

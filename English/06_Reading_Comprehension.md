## The 4-Step Strategic Approach

### Step 1: Scan the Questions First (30 seconds)

```
Before reading the passage, look at ALL questions to identify:
â”œâ”€â”€ WHAT type of questions are asked?
â”œâ”€â”€ WHAT keywords appear in questions?
â”œâ”€â”€ Are there line/paragraph references?
â””â”€â”€ What is the general focus?
```

**Why This Works**: Your brain now knows what to hunt for while reading.

### Step 2: First Pass - Structural Reading (2-3 minutes)

```
Read for STRUCTURE, not detail:
â”œâ”€â”€ What is the MAIN TOPIC? (Usually in first 2-3 sentences)
â”œâ”€â”€ What is the AUTHOR'S STANCE? (For/Against/Neutral)
â”œâ”€â”€ Where are different SUBTOPICS discussed?
â””â”€â”€ Note paragraph transitions
```

**Mark Key Elements**:
- Circle topic shifts
- Underline author's opinion words
- Note examples (don't read in detail)

### Step 3: Attack Questions by Type

```
SEQUENCE STRATEGY:
1. Factual/Detail questions FIRST (quick wins)
2. Vocabulary-in-context NEXT
3. Inference questions THEN
4. Main idea/Title questions LAST
```

### Step 4: Eliminate and Verify

```
For EACH answer choice:
â”œâ”€â”€ Is there EVIDENCE in the passage?
â”œâ”€â”€ Does it MATCH or CONTRADICT?
â”œâ”€â”€ Is it TOO EXTREME or TOO NARROW?

# Question Type Analysis

## Type 1: Main Idea / Central Theme

### How to Identify
```
"The passage is mainly about..."
"The central idea of the passage is..."
"What is the primary purpose..."
"The best title for this passage would be..."
"The author's main argument is..."
```

### Strategy

```
MAIN IDEA = TOPIC + AUTHOR'S POINT ABOUT IT

TOPIC: What is the passage about? (The subject)
POINT: What does the author SAY about it? (The claim)

Example:
Topic: Climate change
Point: It requires immediate global action
Main Idea: The urgent need for global action on climate change
```

### Common Traps

| Trap | Description |
|------|-------------|
| **Too Narrow** | Answer covers only ONE paragraph |
| **Too Broad** | Answer is general, not specific to passage |
| **Mentioned but Minor** | It's in the passage but not the main focus |
| **Opposite** | Reverses the author's actual stance |

### Elimination Technique

```
MUST INCLUDE: The main subject + author's stance
ELIMINATE IF: Only about a detail, example, or minor point
```

---

## Type 2: Detail / Factual Questions

### How to Identify
```
"According to the passage..."
"The author states that..."
"Which of the following is mentioned..."
"The passage indicates that..."
"As per the passage..."
```

### Strategy

```
1. Locate the relevant section (use keywords from question)
2. Read 2-3 lines before and after the reference
3. Match answer to passage (not your knowledge!)
4. Watch for paraphrasing - same meaning, different words
```

### Common Traps

| Trap | Description |
|------|-------------|
| **Not Stated** | Seems logical but isn't in the passage |
| **Distortion** | Uses passage words but changes meaning |
| **Too Extreme** | Uses absolute words (never, always, only) |
| **Right words, wrong context** | Words are from passage but different meaning |

---

## Type 3: Inference Questions

### How to Identify
```
"It can be inferred that..."
"The passage suggests/implies that..."
"Based on the passage, one can conclude..."
"The author would most likely agree that..."
"It may be reasonably concluded that..."
```

### Strategy

```
INFERENCE â‰  Wild Guess
INFERENCE = What MUST be true based on what IS stated

Steps:
1. Find the relevant section
2. Identify what is EXPLICITLY stated
3. Choose the answer that LOGICALLY follows
4. Avoid answers that require OUTSIDE information
```

### The Inference Test

```
Ask: "If the passage is true, MUST this answer be true?"

Too Weak: "might be true" âŒ
Just Right: "must be true based on passage" âœ…
Too Strong: "definitely true universally" âŒ
```

### Common Traps

| Trap | Description |
|------|-------------|
| **Over-inference** | Goes too far beyond the passage |
| **Common knowledge** | True in general but not from passage |
| **Reversal** | Opposite of what passage implies |

## Type 4: Vocabulary in Context

### How to Identify
```
"The word '___' in line X most nearly means..."
"In the context of the passage, '___' refers to..."
"The author uses '___' to mean..."
```

### Strategy

```
1. IGNORE what you think the word means
2. Read the SENTENCE containing the word
3. Look for CONTEXT CLUES:
   â”œâ”€â”€ Definition clues: "X, which means Y"
   â”œâ”€â”€ Example clues: "such as," "for instance"
   â”œâ”€â”€ Contrast clues: "but," "however," "unlike"
   â”œâ”€â”€ Synonym clues: words with similar meaning nearby
4. Substitute each answer choice into the sentence
5. Choose the one that maintains the meaning
```

### Example Walkthrough

```
Passage: "His facile approach to complex problems made colleagues suspicious."

Question: "Facile" most nearly means:
A) Easy
B) Superficial
C) Skillful
D) Friendly

Context clue: "made colleagues suspicious" = negative connotation
"Easy" can be positive; "Skillful" and "Friendly" are positive
"Superficial" is negative and fits the suspicion

Answer: B) Superficial
```

## Type 6: Logical Structure Questions

### How to Identify
```
"The passage is organized by..."
"The function of paragraph 2 is to..."
"The author develops the argument by..."
"The relationship between paragraph X and Y is..."
```

### Common Structures

| Structure | Description |
|-----------|-------------|
| **Problem-Solution** | States problem, then solution |

# Speed Reading Techniques

## Technique 1: Chunking

```

# Common Passage Types in Competitive Exams

## 1. Scientific/Technical Passages

```
Features: Factual, objective, data-heavy
Focus on: Main finding, methodology, conclusions
Trap: Getting lost in technical details
Strategy: Focus on "so what" - the significance
```

## 2. Social Science Passages

```
Features: Theory-based, multiple viewpoints
Focus on: Author's stance, different perspectives
Trap: Confusing author's view with cited views
Strategy: Track who says what
```

## 3. Opinion/Editorial Passages

```
Features: Strong stance, persuasive language
Focus on: Author's argument, supporting evidence
Trap: Agreeing/disagreeing based on your opinion
Strategy: Identify author's logic chain
```

## 4. Comparative Passages

```
Features: Two or more views presented
Focus on: Similarities, differences, relationships
Trap: Attributing wrong view to wrong source
Strategy: Make a quick comparison chart mentally
```

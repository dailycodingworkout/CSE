## 1. The Root Word System

### The DNA of Words

Every complex English word is built from smaller meaningful parts. Master the parts, and you can decode any word.

```
WORD = PREFIX + ROOT + SUFFIX

Example: UNCOMFORTABLE
UN (not) + COMFORT (ease) + ABLE (capable of) = Not capable of ease
```

### ðŸŽ¯ The Synonym Selection Strategy

When choosing synonyms in exams:
1. **Check TONE:** Is the word positive, negative, or neutral?
2. **Check INTENSITY:** Is it mild or strong?
3. **Check CONTEXT:** Does it fit the sentence meaning?

```
Example: "The politician gave a _____ speech."
Options: (a) eloquent (b) garrulous (c) verbose (d) loquacious

Analysis:
- eloquent = positively fluent, persuasive âœ…
- garrulous = negatively talkative âŒ
- verbose = using too many words (negative) âŒ
- loquacious = talkative (neutral/negative) âŒ

Answer: (a) eloquent
```

## 7. Commonly Confused Words

### Words That Sound Similar

| Pair | Meanings | Example |
|------|----------|---------|
| **Accept / Except** | Accept = receive; Except = exclude | "I accept all except him." |
| **Advice / Advise** | Advice = noun; Advise = verb | "I advise you to take my advice." |
| **Affect / Effect** | Affect = verb (influence); Effect = noun (result) | "The effect affects everyone." |
| **Allusion / Illusion** | Allusion = reference; Illusion = false image | "An allusion, not an illusion." |
| **Altar / Alter** | Altar = religious table; Alter = change | "He altered the altar." |
| **Assent / Ascent** | Assent = agree; Ascent = climb | "They gave assent for the ascent." |
| **Beside / Besides** | Beside = next to; Besides = in addition | "He sat beside me; besides, he's my friend." |
| **Breath / Breathe** | Breath = noun; Breathe = verb | "Take a deep breath and breathe." |
| **Canvas / Canvass** | Canvas = cloth; Canvass = survey | "They canvassed votes on canvas posters." |
| **Censor / Censure** | Censor = examine; Censure = criticize | "They censored and censured the article." |
| **Cite / Site / Sight** | Cite = quote; Site = place; Sight = view | "Cite the site with a beautiful sight." |
| **Coarse / Course** | Coarse = rough; Course = path/curriculum | "A coarse course of study." |
| **Complement / Compliment** | Complement = complete; Compliment = praise | "Her skills complement the team. That's a compliment." |
| **Council / Counsel** | Council = committee; Counsel = advice/lawyer | "The council sought legal counsel." |
| **Desert / Dessert** | Desert = dry land (or abandon); Dessert = sweet dish | "Don't desert your dessert in the desert." |
| **Device / Devise** | Device = noun (gadget); Devise = verb (plan) | "He devised a new device." |
| **Elicit / Illicit** | Elicit = draw out; Illicit = illegal | "Elicit information about illicit activities." |
| **Eminent / Imminent / Immanent** | Eminent = famous; Imminent = about to happen; Immanent = inherent | "An eminent danger is imminent." |
| **Fair / Fare** | Fair = just/event; Fare = ticket price | "A fair fare at the fair." |
| **Hoard / Horde** | Hoard = accumulate; Horde = crowd | "A horde hoards gold." |
| **Loose / Lose** | Loose = not tight; Lose = misplace | "Don't lose that loose screw." |
| **Moral / Morale** | Moral = ethics; Morale = spirit | "High moral standards boost morale." |
| **Personal / Personnel** | Personal = individual; Personnel = staff | "Personal issues affect personnel." |
| **Practicable / Practical** | Practicable = can be done; Practical = sensible | "A practical and practicable plan." |
| **Principal / Principle** | Principal = main/head; Principle = rule | "The principal stated the principle." |
| **Stationary / Stationery** | Stationary = not moving; Stationery = paper | "The stationery is stationary." |
| **Than / Then** | Than = comparison; Then = time | "It was better than before, then it improved." |
| **Weather / Whether** | Weather = climate; Whether = if | "Whether the weather permits..." |
| **Who's / Whose** | Who's = who is; Whose = possessive | "Who's the one whose bag is this?" |
| **Your / You're** | Your = possessive; You're = you are | "You're taking your time." |

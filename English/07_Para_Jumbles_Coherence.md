# The 5 Connection Types

## 1. Pronoun Reference Links

**Rule**: A pronoun (he, she, it, they, this, that, these, those, such, etc.) must refer to a noun introduced earlier.

```
SENTENCE A: "The scientist conducted the experiment."
SENTENCE B: "She recorded surprising results."

Connection: "She" refers to "scientist" â†’ A must come before B
```

### Key Pronouns to Watch

| Pronoun | Refers Back To |
|---------|---------------|
| he, she, they | A person/people mentioned earlier |
| it, this, that | A thing/idea mentioned earlier |
| these, those | Multiple things mentioned earlier |
| such | Something of that type |
| the same | Previously mentioned item |
| former, latter | First/second of two mentioned items |
| both, neither, either | Two items mentioned earlier |

### Practice Example

```
P: The experiment failed.
Q: This was unexpected.
R: The researchers had predicted success.
S: They were disappointed.

Analysis:
- "This" in Q refers to something â†’ needs P before it (the failure)
- "They" in S refers to people â†’ needs R before it (researchers)

Logical order: R â†’ P â†’ Q â†’ S
(Researchers predicted success â†’ experiment failed â†’ this was unexpected â†’ they were disappointed)
```

## 2. Article/Determiner Links

**Rule**: "A/An" introduces NEW information; "The" refers to KNOWN information.

```
SENTENCE A: "A man entered the room."
SENTENCE B: "The man was wearing a hat."

Connection: "The man" means we already know about him â†’ A before B
```

### The A/An â†’ The Pattern

```
First mention: "A/An + noun"
Subsequent mentions: "The + noun"

Example sequence:
"A company was founded in 2020." (introduction)
"The company grew rapidly." (known reference)
"The CEO announced expansion." (specific CEO of that company)
```

## 3. Transition/Connector Links

**Rule**: Transition words show logical relationships between sentences.

### Transition Word Categories

| Category | Words | Use |
|----------|-------|-----|
| **Addition** | also, moreover, furthermore, in addition, besides, additionally | Adds more of the same |
| **Contrast** | but, however, yet, nevertheless, on the other hand, conversely, whereas, although | Shows opposition |
| **Cause-Effect** | therefore, thus, hence, consequently, as a result, so, because, since | Shows result |
| **Example** | for example, for instance, such as, to illustrate, specifically | Provides examples |
| **Sequence** | first, second, then, next, finally, subsequently, previously | Shows order |
| **Comparison** | similarly, likewise, in the same way, just as | Shows similarity |
| **Conclusion** | in conclusion, finally, ultimately, in summary, to sum up | Ends discussion |
| **Emphasis** | indeed, in fact, certainly, clearly, obviously | Strengthens point |

### Using Transitions to Order

```
If a sentence starts with "However" or "But":
â†’ The PREVIOUS sentence must present the contrasting view

If a sentence starts with "Therefore" or "Thus":
â†’ The PREVIOUS sentence must contain the cause

If a sentence starts with "For example":
â†’ The PREVIOUS sentence must contain the general statement

If a sentence starts with "Finally" or "In conclusion":
â†’ It's likely the LAST sentence
```

## 4. Logical/Chronological Links

**Rule**: Ideas follow logical sequence - general to specific, cause to effect, chronological order.

### Common Logical Patterns

| Pattern | Sequence |
|---------|----------|
| **General â†’ Specific** | Main idea â†’ Details â†’ Examples |
| **Problem â†’ Solution** | State problem â†’ Discuss implications â†’ Present solution |
| **Cause â†’ Effect** | Cause occurs â†’ Effect happens |
| **Chronological** | Earlier event â†’ Later event |
| **Question â†’ Answer** | Question posed â†’ Answer provided |
| **Theory â†’ Evidence** | Claim made â†’ Supporting evidence |

### Example

```

Q: Respiratory diseases have increased by 40%.
R: Therefore, governments are implementing strict regulations.
S: For instance, many cities now limit vehicle emissions.

Logical order: P â†’ Q â†’ R â†’ S
(General problem â†’ Specific effect â†’ Solution action â†’ Example of action)

# Identifying Opening and Closing Sentences

## Opening Sentence Characteristics

A sentence is likely to be FIRST if it:

```
1. Introduces a topic with general statement
2. Provides background or context
3. Contains a definition
4. Has no pronouns referring to unknown nouns
5. Uses "A/An" to introduce a new subject
6. States a historical origin or beginning
7. Poses a question that the passage will answer
```

### Examples of Opening Sentences

```
âœ“ "Climate change is one of the most pressing challenges of our time."
âœ“ "In 1969, the first humans landed on the moon."
âœ“ "What makes a great leader?"
âœ“ "A new study has revealed surprising findings about sleep."
```

## Closing Sentence Characteristics

A sentence is likely to be LAST if it:

```
1. Contains "in conclusion," "finally," "thus," "therefore"
2. Summarizes the main point
3. Provides a recommendation or future outlook
4. Answers a question posed earlier
5. Makes a general concluding statement
6. Has no forward-pointing references
```

### Examples of Closing Sentences

```
âœ“ "In conclusion, early intervention is key to success."
âœ“ "Only time will tell whether this approach will work."
âœ“ "Thus, the importance of education cannot be overstated."
âœ“ "These findings have significant implications for future research."

# Common Traps and How to Avoid Them

## Trap 1: Misleading Transitions

```
Not every "however" starts a contrast with the immediately preceding sentence.
Sometimes the contrast is with an idea mentioned earlier.

STRATEGY: Check if the "however" makes logical sense with the proposed previous sentence.
```

## Trap 2: Multiple Valid-Looking Pairs

```
Sometimes the same sentence could pair with two different sentences.

STRATEGY: Use other constraints to narrow down. Check with the given options.
```

## Trap 3: Similar Topic but Wrong Subtopic

```
Two sentences might discuss related topics but belong in different parts.

STRATEGY: Pay attention to the specific aspect discussed, not just the general topic.
```

## Trap 4: Attractive Distractors

```
An option might start correctly but have errors later.

STRATEGY: Verify the ENTIRE sequence, not just the beginning.
```

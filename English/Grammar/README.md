## 3. Subject-Verb Agreement

### The Atomic Truth: **Singular Subject = Singular Verb; Plural Subject = Plural Verb**

### The Core Rules

#### Rule 1: Identify the TRUE Subject

**Trap:** Words between subject and verb don't change agreement.

```

â†’ "list" is singular â†’ "is" (ignore "items")

**Key Phrases to Ignore:**

- together with  
- as well as
- including
- besides
- in addition to

```

"The professor, as well as the students, WAS present." 
```

#### Rule 2: Either...Or / Neither...Nor

**The Proximity Rule:** Verb agrees with the CLOSER subject.

```
"Either the teacher OR the students ARE responsible."
"Neither the students NOR the teacher IS responsible."
```

#### Rule 3: Collective Nouns

| Situation | Number | Example |
|-----------|--------|---------|
| Acting as ONE unit | Singular | "The team IS winning." |
| Acting as INDIVIDUALS | Plural | "The team ARE arguing among themselves." |

**Common Collective Nouns:** team, jury, committee, family, audience, staff, public, government

#### Rule 4: Indefinite Pronouns

| Always Singular | Always Plural | Depends on Context |
|----------------|---------------|-------------------|
| each, every, everyone | both, few, many | all, some, most |
| everybody, anyone | several, others | none, any, more |
| somebody, nobody | | half, part |
| nothing, either, neither | | percentage, fraction |

```
"Everyone HAS submitted." (Singular)
"Both HAVE submitted." (Plural)
"Some of the water IS spilled." (Water = uncountable = singular)
"Some of the books ARE missing." (Books = countable = plural)
```

#### Rule 5: Subjects Joined by "AND"

**Generally Plural:**
```
"The manager and the assistant ARE here."
```

**Exception - Same entity or idea:**
```
"Bread and butter IS my breakfast." (One item)
"The director and producer IS John." (Same person)
"Slow and steady WINS the race." (One concept)
```

#### Rule 6: Titles, Names, Numbers

**Always Singular (treated as one unit):**
```
"The Times of India IS a famous newspaper."
"Two thousand rupees IS a lot of money."
"Fifty kilometers IS a long distance."
```

#### Rule 7: "The number of" vs. "A number of"

```
"THE number of students IS increasing." (Singular - refers to the number)
"A number of students ARE absent." (Plural - means "many")
```

#### Rule 8: Here/There + Be

**Verb agrees with the subject that FOLLOWS:**
```
"Here IS the book." (book = singular)
"There ARE many options." (options = plural)
```

3. Strip away prepositional phrases
4. Check: Subject singular? â†’ Verb singular. Subject plural? â†’ Verb plural.

## 6. Voice (Active/Passive)

### The Atomic Truth: **Active = Doer acts; Passive = Receiver is acted upon**

### Basic Transformation

```
ACTIVE:  Subject (Doer) + Verb + Object (Receiver)
PASSIVE: Subject (Receiver) + Be + V3 + by + Object (Doer)
```

### Voice Conversion Table

| Tense | Active | Passive |
|-------|--------|---------|
| Simple Present | writes | is written |
| Present Continuous | is writing | is being written |
| Present Perfect | has written | has been written |
| Simple Past | wrote | was written |
| Past Continuous | was writing | was being written |
| Past Perfect | had written | had been written |
| Simple Future | will write | will be written |
| Future Perfect | will have written | will have been written |

### Step-by-Step Conversion

**Active â†’ Passive:**
1. Object becomes Subject
2. Subject becomes Object (with "by")
3. Add appropriate form of "be"
4. Change verb to V3 (past participle)

**Example:**
```
Active: "The team COMPLETED the project."
        Subject   Verb        Object

Passive: "The project WAS COMPLETED by the team."
         Subject      Be + V3      by + Object
```

### Special Cases

**Case 1: Sentences with Two Objects**
```
Active: "She gave ME a book."
         S    V   IO   DO

Passive (using IO): "I was given a book by her."
Passive (using DO): "A book was given to me by her."
```

**Case 2: Questions in Passive**
```
Active: "Did the company launch the product?"
Passive: "Was the product launched by the company?"
```

**Case 3: Modals**
```
Active: "They CAN solve this problem."
Passive: "This problem CAN BE SOLVED by them."

Structure: Modal + be + V3
```

**Case 4: Imperatives**
```
Active: "Open the door."
Passive: "Let the door be opened."
         OR "You are requested to open the door."
```

**Case 5: Infinitives**
```
Active: "I want someone to help me."
Passive: "I want to be helped."
```

### When to Use Passive Voice

1. **Doer is unknown:** "My wallet WAS STOLEN."
2. **Doer is unimportant:** "The bridge WAS BUILT in 1990."
3. **Scientific writing:** "The sample WAS HEATED to 100Â°C."
4. **Formal writing:** "Applications ARE INVITED for..."
5. **Focus on action:** "The match WAS WON by India."

### ðŸŽ¯ Quick Identification

**Passive Voice Markers:**
- is/am/are/was/were/been/being + V3
- by + agent (optional)

**Check:** Can you add "by someone"? If yes, it's passive.

**Trap 1: "By" doesn't always indicate passive**
```
"He sat BY the window." (Preposition of place, not passive)
"The book BY John is famous." (Author, not passive)

**Trap 2: Intransitive verbs cannot be passivized**

## 9. Conditionals

### The Atomic Truth: **Conditionals = IF (condition) â†’ THEN (result)**

### The Four Types

#### Type 0: Universal Truth/Facts
```
Structure: If + Present Simple, Present Simple
Usage: General truths, scientific facts

"If you heat water to 100Â°C, it boils."
"If it rains, the ground gets wet."
```

#### Type 1: Real/Likely Condition (Future)
```
Structure: If + Present Simple, will + V1
Usage: Real possibilities, likely situations

"If you study hard, you WILL pass."
"If it rains tomorrow, I WILL stay home."
```

#### Type 2: Unreal/Hypothetical (Present)
```
Structure: If + Past Simple, would + V1
Usage: Imaginary present, unlikely situations

"If I WERE rich, I WOULD travel the world."
"If she STUDIED harder, she WOULD score better."

Note: "WERE" is used for all persons (If I were, If he were)
```

#### Type 3: Unreal Past (Impossible Now)
```
Structure: If + Past Perfect, would have + V3
Usage: Imaginary past, regrets, impossible changes

"If I HAD STUDIED, I WOULD HAVE PASSED."
"If she HAD LEFT earlier, she WOULD HAVE CAUGHT the train."
```

### Mixed Conditionals

**Past Condition â†’ Present Result:**
```
"If I HAD STUDIED medicine, I WOULD BE a doctor now."
(Type 3 condition, Type 2 result)
```

**Present Condition â†’ Past Result:**
```
"If she WEREN'T so lazy, she WOULD HAVE FINISHED by now."
(Type 2 condition, Type 3 result)
```

### Alternative Conditional Words

| Instead of "IF" | Usage | Example |
|----------------|-------|---------|

| Type | With If | Inverted |

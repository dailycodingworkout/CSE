## Context Clue Types in Detail

### Type 1: Definition/Restatement Clues

The sentence defines or restates the blank.

```
Example: "The _____ leader, known for his strict discipline and authoritarian methods, controlled every aspect of the organization."

Clue: "strict discipline and authoritarian methods"
Answer: tyrannical/despotic

The clue after the comma defines what kind of leader.
```

### Type 2: Example Clues

The sentence gives examples of what the blank means.

```
Example: "_____ animals, such as lions, tigers, and wolves, hunt other animals for food."

Clue: "such as lions, tigers, and wolves"
Answer: Carnivorous/Predatory

The examples are all meat-eaters.
```

### Type 3: Contrast Clues

The sentence shows contrast - the blank is opposite to another word.

```
Example: "Unlike his _____ brother, who always spoke his mind, John was reserved and quiet."

Clue: "Unlike... reserved and quiet"
Answer: outspoken/candid

The blank is opposite to "reserved and quiet."
```

**Signal words**: unlike, but, however, whereas, while, although, despite, instead of, on the other hand

### Type 4: Similarity Clues

The sentence shows similarity - the blank is similar to another word.

```
Example: "The patient made a _____ recovery, and the doctors were equally pleased with his rapid improvement."

Clue: "rapid improvement"
Answer: speedy/swift/quick

The blank should mean something similar to "rapid."
```

**Signal words**: and, similarly, just as, also, likewise, in the same way

### Type 5: Cause-Effect Clues

The blank is either the cause or effect of something stated.

```
Example: "Because the evidence was _____, the case was dismissed due to lack of proof."

Clue: "case was dismissed due to lack of proof"
Answer: insufficient/inadequate

The effect (dismissal) helps identify the cause (insufficient evidence).
```

## Positive/Negative Tone Strategy

### Step 1: Determine the Required Tone

```
POSITIVE INDICATORS: praise, success, improvement, advantage
NEGATIVE INDICATORS: criticism, failure, decline, disadvantage
NEUTRAL INDICATORS: factual statements, descriptions
```

### Step 2: Match Options to Tone

```
Example: "Despite his _____ appearance, the generous philanthropist helped thousands."

Tone clue: "Despite" + "generous philanthropist helped"
Required: NEGATIVE appearance (contrasted with positive action)
Answer: disheveled/unkempt/scruffy (negative appearance)
NOT: elegant/impressive (positive - wrong contrast)
```

# Double Blank Questions

## The 2-Blank Strategy

### Method 1: Work One Blank at a Time

```
Step 1: Try to solve the EASIER blank first
Step 2: Eliminate options where that blank doesn't work
Step 3: Verify the remaining blank(s) with reduced options
```

```

â”œâ”€â”€ Same direction (both positive or both negative)

## Common Double Blank Patterns

### Pattern 1: Contrast Pattern (despite, although, but)

```

Rule: Blanks should be OPPOSITE

### Pattern 2: Parallel Pattern (and, similarly, both)

```

Rule: Blanks should be SIMILAR/COMPLEMENTARY

Example answer: captivating, memorable
```

### Pattern 3: Cause-Effect Pattern (so, therefore, because)

```

Rule: First blank CAUSES the second

Example answer: exquisite, glowing

## Grammar-Based Cloze Clues

### Clue 1: Article + Blank

```
"A + _____ + noun" â†’ The blank needs an adjective starting with consonant sound
"An + _____ + noun" â†’ The blank needs an adjective starting with vowel sound

Example: "It was an _____ experience."
Options: incredible (âœ“), remarkable (âœ— - needs "a")
```

### Clue 2: Preposition Requirements

```
Some words require specific prepositions:
"depend + on" not "depend + from"
"different + from" not "different + than"

Example: "The result depends _____ many factors."
Answer: on
```

### Clue 3: Subject-Verb Agreement

```
If the subject is singular, the verb must be singular.
If the subject is plural, the verb must be plural.

Example: "The group of students _____ excited."
Answer: was (group = singular)
```

### Clue 4: Parallel Structure

```
Items in a list should have the same form.

Example: "She enjoys reading, _____, and swimming."
Options: to write (âœ—), writing (âœ“), writes (âœ—)
```

## Semantic Cloze Clues

### Clue 1: Collocation (Words That Go Together)

```
Certain words naturally pair:
"make a decision" (not "do a decision")
"heavy rain" (not "strong rain")
"commit a crime" (not "make a crime")

Example: "The committee will _____ a final decision tomorrow."
Answer: make (not take, do, or have)
```

### Common Collocations to Know

| Correct | Incorrect |
|---------|-----------|

| make a mistake | do a mistake |
| do homework | make homework |
| do business | make business |
| take responsibility | make responsibility |
| take a break | make a break |
| have a meeting | make a meeting |
| pay attention | give attention |
| draw a conclusion | make a conclusion |

### Clue 2: Logical Connectors

```
The connector tells you the relationship:
"however" â†’ contrast follows
"therefore" â†’ result follows
"moreover" â†’ additional similar point follows

Example: "The evidence was strong; _____, the defendant was convicted."
Answer: therefore/consequently
```

### Clue 3: Semantic Field

```
Words in a passage usually belong to the same semantic field.

If discussing "economy":
Expect: growth, decline, investment, profit, market
Don't expect: photosynthesis, molecules, velocity
```

## Common Trap Patterns

### Trap 1: Similar-Looking Options

```

```

```

```

```

```

```

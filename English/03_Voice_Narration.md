# Part 1: Active & Passive Voice

## ğŸ§  The Mental Model: The Camera Angle

```
ACTIVE VOICE = Camera on the DOER
"The chef prepared the meal."
[Focus: Chef is doing something]

PASSIVE VOICE = Camera on the RECEIVER
"The meal was prepared by the chef."
[Focus: Something is being done to the meal]
```

## WHY Use Passive Voice?

| Reason | Example |
|--------|---------|
| Doer unknown | "My wallet **was stolen**." (don't know who) |
| Doer unimportant | "The bridge **was built** in 1950." (focus on bridge) |
| Doer obvious | "The criminal **was arrested**." (obviously by police) |
| Formal/Scientific writing | "The experiment **was conducted**." |
| To be diplomatic | "Mistakes **were made**." (avoids blame) |

---

## The Transformation Formula

### Basic Structure
```
ACTIVE:  Subject (doer) + Verb + Object (receiver)
PASSIVE: Object (receiver) + be + V3 (past participle) + by Subject (doer)
```

### Step-by-Step Process

```
Step 1: Identify Subject, Verb, and Object in active sentence
Step 2: Move Object to subject position
Step 3: Change verb to "be" + past participle (V3)
Step 4: Add "by" + original subject (optional if obvious)
```

### Example Walkthrough
```
ACTIVE: "The dog bit the man."

       Subject    Verb   Object

PASSIVE: "The man was bitten by the dog."
            â†‘         â†‘           â†‘
         Object    be+V3    by+Subject
```

## Special Passive Constructions

### 1. Two-Object Verbs (Ditransitive)

Some verbs have two objects: Indirect Object (person) and Direct Object (thing)

```
ACTIVE: She gave me a book.
            â†‘    â†‘
           IO   DO

Two possible passives:
PASSIVE 1: I was given a book (by her).     [IO becomes subject]
PASSIVE 2: A book was given to me (by her). [DO becomes subject]
```

**Common two-object verbs**: give, tell, show, teach, send, offer, pay, ask, promise, allow

### 2. Phrasal Verbs

Keep the preposition attached:
```
ACTIVE: They laughed at him.
PASSIVE: He was laughed at. âœ… (not "He was laughed")

ACTIVE: Someone broke into the house.
PASSIVE: The house was broken into.
```

### 3. "Let" in Passive

```
ACTIVE: Let him go.
PASSIVE: Let him be allowed to go. 
         OR: He should be let go.
```

### 4. Imperative Sentences

```
ACTIVE: Close the door.
PASSIVE: Let the door be closed.
         OR: You are requested to close the door. (polite)

ACTIVE: Do not touch this.
PASSIVE: Let this not be touched.
         OR: This should not be touched.
```

### 5. Infinitives

```
ACTIVE: I want someone to help me.
PASSIVE: I want to be helped.

ACTIVE: He is to finish the work.
PASSIVE: The work is to be finished by him.
```

## ğŸš¨ Common Traps in Voice

### Trap 1: Intransitive Verbs Cannot Be Made Passive

**Intransitive verbs** have no object â†’ No passive possible

```
âŒ "He was laughed." (laugh needs an object with preposition)
âœ… "He was laughed at."

âŒ "The accident was happened." (happen = intransitive, no object)
âœ… "The accident happened."
```

**Common intransitive verbs** (no passive): happen, occur, arrive, die, sleep, go, come, fall, rise, appear, exist

### Trap 2: "By" Is Not Always Needed

When the doer is:
- Unknown: "The window was broken."
- Obvious: "He was arrested." (obviously by police)
- Unimportant: "The letter was posted."

### Trap 3: Stative Verbs in Passive

Some state verbs don't typically form meaningful passives:
```
âŒ "The answer is been known by me."
âœ… "I know the answer." (active preferred)
```

### Trap 4: "Get" Passive

Informal passive using "get" instead of "be":
```
"He got injured." = "He was injured."
"They got married." = "They were married."
```

## Transformation by Sentence Type

### Type 1: Statements (Assertive)

**Direct**: He said, "I am going home."
**Indirect**: He said that he was going home.

```
Structure: Reporting verb + (that) + subject + verb (backshifted)
```

**"Say" vs "Tell"**:
```
SAY: He said that...       (no object, or "to + object")
     He said to me that...

TELL: He told me that...   (always needs an object)
      âŒ He told that...
```

---

### Type 2: Questions (Interrogatives)

#### Yes/No Questions

**Direct**: She asked, "Are you coming?"
**Indirect**: She asked if/whether I was coming.

```
Structure: Asked + if/whether + subject + verb (statement order)
```

**Note**: Remove the question mark; change to statement word order.

#### Wh-Questions

**Direct**: He asked, "Where do you live?"
**Indirect**: He asked where I lived.

```
Structure: Asked + wh-word + subject + verb (statement order)
```

**Examples**:
```
"What is your name?" â†’ He asked what my name was.
"When will you return?" â†’ He asked when I would return.
"Why are you late?" â†’ He asked why I was late.
```

---

### Type 3: Commands & Requests (Imperatives)

**Direct**: He said, "Go away."
**Indirect**: He ordered me to go away.

**Direct**: She said, "Please help me."
**Indirect**: She requested me to help her.

```
Structure: 
COMMAND: ordered/commanded + object + to + V1
REQUEST: requested/asked + object + to + V1
```

**Negative**:
```
"Don't touch it." â†’ He told me not to touch it.
"Please don't go." â†’ She requested me not to go.
```

---

### Type 4: Exclamations

**Direct**: She said, "What a beautiful painting!"
**Indirect**: She exclaimed that it was a very beautiful painting.

**Direct**: He said, "Hurrah! We have won!"
**Indirect**: He exclaimed with joy that they had won.

```
Structure: exclaimed (with joy/sorrow/surprise) + that + ...
```

---

### Type 5: Suggestions

**Direct**: He said, "Let's go for a walk."
**Indirect**: He suggested that we should go for a walk.
            OR: He suggested going for a walk.

## ğŸš¨ EXCEPTIONS: When NOT to Backshift

### 1. Universal Truths
```
DIRECT: Teacher said, "The earth revolves around the sun."
INDIRECT: Teacher said that the earth revolves around the sun.
(No backshift - still true)
```

### 2. Historical Facts
```
DIRECT: He said, "World War II ended in 1945."
INDIRECT: He said that World War II ended in 1945.
```

### 3. Reporting Verb in Present/Future Tense
```
DIRECT: He says, "I am busy."
INDIRECT: He says that he is busy. (no backshift)
```

### 4. When the Situation Is Still True
```
DIRECT: She said, "I live in Mumbai."
INDIRECT: She said that she lives in Mumbai. (if still true)
          She said that she lived in Mumbai. (if uncertain or changed)
```

### 5. Conditionals (Type 2 & 3)
```
DIRECT: "If I were you, I would accept."
INDIRECT: He said that if he were me, he would accept.
(No change in Type 2 conditional)

DIRECT: "If I had known, I would have come."
INDIRECT: He said that if he had known, he would have come.
(No change in Type 3 conditional)
```

### 6. Must (for logical deduction)
```
DIRECT: "He must be at home."
INDIRECT: She said that he must be at home. (no change)
(But "must" for obligation â†’ "had to")
```

## üî• Important Formulas & Shortcuts

### Quantitative Aptitude Quick Formulas

#### Percentage
- To convert fraction to percentage: (Fraction √ó 100)%
- Percentage increase/decrease: [(New Value - Original Value) / Original Value] √ó 100

#### Simple Interest
- SI = (P √ó R √ó T) / 100
- Amount = P + SI

#### Compound Interest
- CI = P[(1 + R/100)^T - 1]
- Amount = P(1 + R/100)^T

#### Time and Work
- Work = Rate √ó Time
- If A can do work in 'a' days and B in 'b' days, together they can do in: (a √ó b)/(a + b) days

#### Time, Speed and Distance
- Speed = Distance / Time
- Average Speed = Total Distance / Total Time

#### Profit and Loss
- Profit% = (Profit / CP) √ó 100
- Loss% = (Loss / CP) √ó 100
- SP = CP √ó (100 + Profit%) / 100

### Reasoning Shortcuts

#### Syllogism Tricks
- Learn Venn diagram method
- Practice "All, Some, No, Not" combinations
- Master complement rules

#### Coding-Decoding Patterns
- Letter shifting
- Number patterns
- Position-based coding

### English Language Tips

#### Reading Comprehension Strategy
1. Read the passage quickly first
2. Identify the main theme

#### Grammar Rules Memory

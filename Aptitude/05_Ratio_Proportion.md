## üìö Core Concepts

### What is a Ratio?

**Ratio** = Comparison of two quantities of the same kind

```
a : b = a/b (read as "a is to b")
```

**üí° Analogy**: Ratio is like a recipe - "2 parts flour to 1 part sugar" = 2:1

**Key Properties**:
- Order matters: 2:3 ‚â† 3:2
- No units (pure number)
- Both quantities must be positive
- Can be scaled: 2:3 = 4:6 = 6:9

---

### What is Proportion?

**Proportion** = Equality of two ratios

```
a : b = c : d  OR  a/b = c/d

Written as: a : b :: c : d
Read as: "a is to b as c is to d"
```

**Key Terms**:
- a, d = Extremes (outer terms)
- b, c = Means (inner terms)

---

## üìê Fundamental Formulas

### Cross Multiplication Rule

If a : b = c : d, then:
```
a √ó d = b √ó c   (Product of extremes = Product of means)
```

---

### Types of Proportion

**1. Mean Proportion (Geometric Mean)**
If a : x = x : b, then x is mean proportion
```
x¬≤ = ab
x = ‚àö(ab)
```

**2. Third Proportion**
If a : b = b : c, then c is third proportion
```
c = b¬≤/a
```

**3. Fourth Proportion**
If a : b = c : d, then d is fourth proportion
```
d = bc/a
```

## üîó Essential Properties of Ratios

### Property 1: Componendo

If a/b = c/d, then:
```
(a+b)/b = (c+d)/d
```

---

### Property 2: Dividendo

If a/b = c/d, then:
```
(a-b)/b = (c-d)/d
```

---

### Property 3: Componendo-Dividendo

If a/b = c/d, then:
```
(a+b)/(a-b) = (c+d)/(c-d)
```

**‚ö° Super useful for competitive exams!**

---

### Property 4: Invertendo

If a/b = c/d, then:
```
b/a = d/c
```

---

### Property 5: Alternendo

If a/b = c/d, then:
```
a/c = b/d
```

---

### Property 6: Addition/Subtraction of Equal Ratios

If a/b = c/d = e/f = k, then:
```
(a+c+e)/(b+d+f) = k

(a-c+e)/(b-d+f) = k (any combination)
```

## üìä Types of Ratio Problems

### Type 1: Finding Ratio

**Example**: A = 30, B = 45. Find A:B
```
A:B = 30:45 = 2:3
```

**‚ö° Always simplify** by dividing both by HCF.

---

### Type 2: Combining Ratios

**When one quantity is common**:
A:B = 2:3 and B:C = 4:5, find A:B:C

```
Make B equal in both:
A:B = 2:3 = 8:12
B:C = 4:5 = 12:15

A:B:C = 8:12:15
```

**‚ö° Trick**: 
```
A:B:C = (A:B √ó B:C first term) : (B common) : (B:C √ó A:B second term)
      = 2√ó4 : 3√ó4 : 3√ó5 = 8:12:15
```

---

### Type 3: Dividing a Quantity in Given Ratio

**Divide N in ratio a:b:c**:
```
First part = N √ó a/(a+b+c)
Second part = N √ó b/(a+b+c)
Third part = N √ó c/(a+b+c)
```

**Example**: Divide 120 in ratio 2:3:5
```
Parts = 120 √ó 2/10, 120 √ó 3/10, 120 √ó 5/10
      = 24, 36, 60
```

---

### Type 4: Finding Original Quantities from Ratio

**Given**: A:B = 3:5, A + B = 160. Find A and B.
```
Total parts = 3 + 5 = 8
A = 160 √ó 3/8 = 60
B = 160 √ó 5/8 = 100
```

---

### Type 5: Ratio Changes

**Given**: Original A:B = 2:3. If A increases by 10 and B decreases by 5, new ratio = 3:4. Find original values.
```
Let A = 2x, B = 3x
(2x + 10)/(3x - 5) = 3/4
4(2x + 10) = 3(3x - 5)
8x + 40 = 9x - 15
x = 55
A = 110, B = 165
```

## ‚öñÔ∏è Direct vs Inverse Proportion

### Direct Proportion

When one increases, other increases proportionally.
```
If A ‚àù B, then A = kB (k = constant)
A‚ÇÅ/A‚ÇÇ = B‚ÇÅ/B‚ÇÇ
```

**Examples**: Distance ‚àù Speed (time constant), Cost ‚àù Quantity

---

### Inverse Proportion

When one increases, other decreases proportionally.
```
If A ‚àù 1/B, then A √ó B = k (constant)
A‚ÇÅ √ó B‚ÇÅ = A‚ÇÇ √ó B‚ÇÇ
A‚ÇÅ/A‚ÇÇ = B‚ÇÇ/B‚ÇÅ
```

**Examples**: Speed ‚àù 1/Time (distance constant), Workers ‚àù 1/Days (work constant)

---

### Combined Proportion

```
If A ‚àù B and A ‚àù 1/C, then A ‚àù B/C
A = k √ó B/C
```

**Example**: Work done ‚àù (Men √ó Days) / Difficulty

## üë• Partnership Problems

### Simple Partnership (Same Time)

If A and B invest for **same time period**:
```
Profit share A : Profit share B = Investment A : Investment B
```

---

### Compound Partnership (Different Times)

If A invests ‚ÇπP‚ÇÅ for T‚ÇÅ months, B invests ‚ÇπP‚ÇÇ for T‚ÇÇ months:
```
Profit share A : Profit share B = P‚ÇÅ√óT‚ÇÅ : P‚ÇÇ√óT‚ÇÇ
```

**Example**: A invests ‚Çπ5000 for 8 months, B invests ‚Çπ6000 for 10 months
```
Ratio = 5000√ó8 : 6000√ó10 = 40000 : 60000 = 2:3
```

---

### Working Partner

If one partner works + invests:
```
Working partner gets salary + share of remaining profit
```

## üìê Variation Problems

### Joint Variation

```
If A varies directly with B and inversely with C:
A ‚àù B/C
A = k √ó B/C

To find k: Use given values
```

**Example**: A = 12 when B = 4, C = 2. Find A when B = 6, C = 3.
```
k = A √ó C/B = 12 √ó 2/4 = 6
New A = 6 √ó 6/3 = 12
```

---

### Chain Rule

For problems involving multiple quantities:
```
A‚ÇÅ/A‚ÇÇ = (all direct ratios) √ó (inverse of all inverse ratios)
```

**Example**: 15 men complete work in 20 days. How many days for 25 men?
```
Men ‚Üë, Days ‚Üì (inverse)
15/25 = D‚ÇÇ/20
D‚ÇÇ = 20 √ó 15/25 = 12 days
```

## üí° Advanced Tricks

### Trick 1: Quick Ratio from Percentages

```
A is 20% more than B
A/B = 120/100 = 6/5
A : B = 6 : 5
```

---

### Trick 2: Finding Individual from Sum/Difference

Given A:B = 3:5 and A - B = 20:
```
Difference in parts = 5 - 3 = 2
Each part = 20/2 = 10
A = 30, B = 50
```

---

### Trick 3: Duplicate/Triplicate Ratios

```
Duplicate ratio of a:b = a¬≤:b¬≤
Triplicate ratio of a:b = a¬≥:b¬≥
Sub-duplicate ratio of a:b = ‚àöa:‚àöb
Sub-triplicate ratio of a:b = ‚àõa:‚àõb
```

---

### Trick 4: Reciprocal Ratio

```
Reciprocal of a:b = b:a = 1/a : 1/b
```

## ‚ö†Ô∏è Edge Cases & Traps

### Trap 1: Same Unit Required
```
‚ùå 2 kg : 500 g = 2:500
‚úÖ 2000 g : 500 g = 4:1
```

### Trap 2: Order Matters
```
A:B = 2:3 means A < B
B:A = 3:2 means B > A
```

### Trap 3: Percentage vs Ratio Points
```
If A is 25% of B, then A:B = 1:4 (NOT 25:100 = 1:4, but different thinking)
If A:B = 1:4, then A is 25% of B, or A is 20% of (A+B)
```

### Trap 4: Addition/Subtraction of Ratios
```
2:3 + 3:4 ‚â† 5:7
Ratios cannot be added like fractions without common entity
```

| Data-heavy | Approximate when exact calculation is slow |

## üî¢ Master Formulas Quick Reference

### Clocks
$$\theta = |30H - 5.5M|$$

### Calendars (Odd Days)
- Ordinary year: 1 odd day
- Leap year: 2 odd days
- Century: 0 (400 yrs), 5 (100 yrs), 3 (200 yrs), 1 (300 yrs)

### Percentage Change
$$\% \text{ Change} = \frac{\text{New} - \text{Old}}{\text{Old}} \times 100$$

### Data Sufficiency Decision
1. Check Statement I alone
2. Check Statement II alone (forget I!)
3. Combine only if neither works alone

### Syllogisms
- All ‚Üí Some (valid)
- Some ‚Üí All (invalid)
- No ‚Üî No (convertible)

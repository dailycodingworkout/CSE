## üìö Part 1: Sentence Completion

### Strategy

1. Read the entire sentence
2. Identify context clues
3. Look for signal words (but, although, therefore)
4. Eliminate incorrect options

---

### Signal Words

| Type | Words | Effect |
|------|-------|--------|
| **Contrast** | but, however, although, despite, yet | Opposite idea follows |
| **Continuation** | and, moreover, furthermore, also | Same idea continues |
| **Cause-Effect** | because, since, therefore, thus | Result follows cause |
| **Example** | for instance, such as, like | Illustration follows |

**Example**:
```
Although he was tired, he _______ the marathon.
Signal word "Although" ‚Üí contrast ‚Üí he finished despite tiredness
Answer: completed/finished
```

## üìö Part 2: Analogies

### What is an Analogy?

A relationship between pairs of words.

**Format**: A : B :: C : D (A is to B as C is to D)

---

### Common Relationship Types

| Relationship | Example |
|--------------|---------|
| **Synonym** | Happy : Joyful :: Sad : Melancholy |
| **Antonym** | Hot : Cold :: Light : Dark |
| **Part to Whole** | Petal : Flower :: Wheel : Car |
| **Category** | Rose : Flower :: Apple : Fruit |
| **Degree** | Warm : Hot :: Cool : Cold |
| **Function** | Pen : Write :: Knife : Cut |
| **Worker : Tool** | Carpenter : Hammer :: Painter : Brush |
| **Object : Action** | Bird : Fly :: Fish : Swim |
| **Cause : Effect** | Fire : Smoke :: Rain : Flood |
| **Male : Female** | Bull : Cow :: Rooster : Hen |
| **Young : Adult** | Puppy : Dog :: Kitten : Cat |
| **Creator : Creation** | Author : Book :: Composer : Symphony |

---

### Solving Strategy

1. Identify the relationship in the first pair
2. Express it as a sentence: "A is the ___ of B"
3. Apply same relationship to find answer
4. Verify the relationship matches

**Example**:
```
Doctor : Hospital :: Teacher : ?
Relationship: Doctor works in Hospital
Teacher works in ‚Üí School
```

## üìö Part 3: Reading Comprehension

### Types of Questions

| Type | What It Tests |
|------|---------------|
| **Main Idea** | Central theme of passage |
| **Detail** | Specific facts stated in passage |
| **Inference** | Conclusions from given info |
| **Vocabulary** | Word meaning in context |
| **Tone/Attitude** | Author's perspective |
| **Assumption** | Underlying beliefs |
| **Strengthen/Weaken** | Effect on argument |

---

### Strategy

1. **Skim first**: Get overall idea (2-3 minutes)
2. **Read questions**: Know what to look for
3. **Detailed read**: Focus on relevant parts
4. **Answer**: Use process of elimination

---

### Inference Questions

Look for:
- Logical conclusions
- NOT directly stated but implied
- Must be supported by passage

**Keywords**: "implies," "suggests," "can be inferred," "most likely"

---

### Main Idea Questions

Look for:
- Thesis statement (often first or last paragraph)
- Recurring theme
- What ties everything together

**Keywords**: "primarily about," "main idea," "best title"

## üìö Part 4: Critical Reasoning

### Argument Structure

```
Premise(s) ‚Üí Conclusion
Evidence ‚Üí Inference
```

**Components**:
- **Premise**: Supporting evidence/facts
- **Conclusion**: What is being argued
- **Assumption**: Unstated belief connecting premise to conclusion

---

### Finding the Conclusion

Look for:
- Keywords: therefore, thus, hence, so, consequently
- The "main point" being made
- What the argument is trying to prove

---

### Finding Assumptions

An assumption:
- Is NOT stated but required for conclusion
- If negated, destroys the argument
- Bridges the gap between premise and conclusion

**Negation Test**: If negating a statement destroys the argument, it's an assumption.

---

### Strengthen/Weaken Questions

**To Strengthen**: Add supporting evidence
**To Weaken**: Attack the assumption or provide counter-evidence

**Approach**:
1. Find the conclusion
2. Identify the assumption
3. Find option that supports/attacks it

---

### Flaw Identification

Common logical flaws:
| Flaw | Description |
|------|-------------|
| **Correlation ‚â† Causation** | A correlates with B doesn't mean A causes B |
| **Hasty Generalization** | Small sample ‚Üí broad conclusion |
| **False Analogy** | Comparing unlike things |
| **Ad Hominem** | Attacking person, not argument |
| **Circular Reasoning** | Conclusion restates premise |
| **False Dichotomy** | Only two options presented (more exist) |
| **Appeal to Authority** | Expert opinion without evidence |
| **Straw Man** | Misrepresenting opponent's position |

---

## üìö Part 5: Vocabulary in Context

### Strategy

1. Read the surrounding sentences
2. Look for context clues (definitions, examples, contrasts)
3. Substitute options and check if meaning is preserved

---

### Context Clue Types

| Type | Signal | Example |
|------|--------|---------|
| **Definition** | is, means, refers to | "Pusillanimous, meaning cowardly, ..." |
| **Example** | such as, like, for instance | "Inclement weather, such as storms and heavy snow, ..." |
| **Contrast** | but, however, unlike | "Unlike her gregarious sister, she was introverted" |
| **Restatement** | or, in other words | "The ephemeral, or short-lived, beauty ..." |

---

## üìö Part 6: Verbal Reasoning Patterns

### Statement-Conclusion

Given statements, determine if conclusion follows.

**Options typically**:
- Only conclusion I follows
- Only conclusion II follows
- Both follow
- Neither follows

---

### Statement-Assumption

Identify what must be true for the statement to hold.

**Key**: The assumption is NECESSARY, not just helpful.

---

### Course of Action

Given a problem situation, identify appropriate action.

**Evaluation criteria**:
- Does it address the problem?
- Is it practical?
- Are there negative consequences?

---

## üí° Quick Tips & Tricks

### Tip 1: Eliminate Extremes

Options with "always," "never," "all," "none" are often wrong.

---

### Tip 2: Parallel Structure

In analogies, maintain same relationship type.

---

### Tip 3: Key Words Matter

Pay attention to qualifiers: some, most, all, few, many, never, always.

---

### Tip 4: Author's Tone

| Tone | Indicators |
|------|------------|
| Positive | praise, admiration, support |
| Negative | criticism, disapproval, concern |
| Neutral | factual, objective, balanced |
| Sarcastic | irony, mockery |

---

### Tip 5: Stay Within Passage

For RC, answers must be supported by passage text.
Don't use outside knowledge.

---

## ‚ö†Ô∏è Common Traps

### Trap 1: Extreme Options
```
Options with "never" or "always" are usually wrong
Real-world situations rarely have absolutes
```

### Trap 2: True but Irrelevant
```
Option may be true but doesn't answer the question
Check what's being asked
```

### Trap 3: Partially Correct
```
Option may be half right but contains an error
Read entire option carefully
```

### Trap 4: Out of Scope
```
Introducing information not in passage
Stay within given content
```

### Trap 5: Misread Qualifiers
```
"Some" vs "All" vs "Most" changes meaning significantly
"Can" vs "Will" vs "Must" also differ
```

## üìù GATE-Level Practice

**Q1**: Complete the analogy: Composer : Symphony :: Architect : ?
```
Relationship: Creator : Creation
Composer creates Symphony
Architect creates ‚Üí Building/Blueprint
```

**Q2**: "Although the evidence was circumstantial, the jury _______ the defendant."
```
Signal "Although" indicates contrast
Circumstantial evidence usually = not enough to convict
But "Although" suggests opposite happened
Answer: convicted
```

**Q3**: The passage states that deforestation leads to soil erosion. Which of the following, if true, would WEAKEN this claim?
```
Look for: Counter-evidence or alternative explanation
Example: "Regions with high deforestation show no change in soil erosion rates"
```

**Q4**: Find the assumption: "Sales increased after the ad campaign, so the campaign was successful."
```
Assumption: The ad campaign caused the sales increase
(Assumes no other factors contributed)
```

**Q5**: Analogy: Painter : Canvas :: ?
```
Painter works ON Canvas
Writer : Paper
Sculptor : Clay
Programmer : Code
```

## üß† Study Strategy

### For GATE (8-10 marks, 4-6 questions)
1. **Must Master**: Subnetting, TCP/UDP, Sliding Window, Routing (60% questions)
2. **High Frequency**: Flow Control, Error Detection, MAC Protocols
3. **Numerical Focus**: Throughput, Delay calculations, IP addressing

### For ESE (10-15 marks)
1. **Theoretical Depth**: Protocol mechanisms, OSI vs TCP/IP comparisons
2. **Conceptual**: Why each layer exists, design decisions
3. **Application**: Real-world scenarios and troubleshooting

### For PSU & Banking
1. **Breadth over Depth**: Cover all topics at basic level
2. **Quick Facts**: Memorize port numbers, protocol functions
3. **Security Focus**: Common attacks, prevention methods

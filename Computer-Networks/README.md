## ðŸ§® Essential Formulas

| Formula | Description |
|---------|-------------|
| $T_{trans} = \frac{L}{R}$ | Transmission Delay (L=packet size, R=bandwidth) |
| $T_{prop} = \frac{d}{v}$ | Propagation Delay (d=distance, v=speed) |
| $RTT = 2 \times T_{prop}$ | Round Trip Time |
| $a = \frac{T_{prop}}{T_{trans}}$ | Efficiency parameter |
| $Efficiency = \frac{1}{1+2a}$ | Stop & Wait efficiency |
| $Throughput = \frac{W}{RTT}$ | Window-based throughput |
| $Subnet_{hosts} = 2^n - 2$ | Usable hosts (n = host bits) |
